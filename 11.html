기능 삽플링 클레임 가져오기 (아래 둘중 하나 골라서 사용)
URL https://api.shopling.co.kr/claim/claim_gather_api.phtml?mode=1&xml_src_url=(아래 xml 데이타가 있는 url 기술) https://api.shopling.co.kr/claim/claim_gather_api.phtml?mode=2 -> 소켓통신으로 바로 xml로 밀어넣을시
※ 필수값이 아니고 해당사항이 없는경우는 미선언 하시면 됩니다.
요청 XML node 반복 field field 설명 Data type format max 필수 비고
request <reqst>
request <apiClaimGather>
request login_id 로그인ID <![CDATA[]]> Y
request company_id 고객사 아이디 Y shopPling 발급
request api_auth_key API 인증코드 Y shopPling 발급
request start_dt 검색시작일자(클레임수집기준일자) yyyymmdd Y 3개월 이내로 검색
request end_dt 검색종료일자(클레임수집기준일자) yyyymmdd Y
request claim_fields 출력 클레임필드명(구분자"," 로 여러개가능) <![CDATA[]]> Y 클레임필드 기술서 참고하여 필요정보 기술
request </apiClaimGather>
request </reqst>
응답 XML node 반복 field field 설명 Data type format max 필수 비고
response <rspns>
response <apiClaimGatherRst>
response login_id 로그인ID <![CDATA[]]> Y
response company_id 고객사 아이디 Y shopPling 발급
response <claimListRst> Y
response Y claim_var_name 클레임정보 필드 변수명 <![CDATA[]]> 클레임필드 기술서 변수 참조
response </claimListRst> Y
response cnt 전체건수 전체 건수
response </apiClaimGatherRst>
response </rspns>

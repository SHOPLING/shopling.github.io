QnA 답변등록 (마지막 수정일시: 2023-08-22 14:23:03)

기능 삽플링에 QnA답변등록 하기 (아래 둘중 하나 골라서 사용)
URL https://api.shopling.co.kr/qna/qna_replyReg_api.phtml?mode=1&xml_src_url=(아래 xml 데이타가 있는 url 기술) https://api.shopling.co.kr/qna/qna_replyReg_api.phtml?mode=2 -> 소켓통신으로 바로 xml로 밀어넣을시
※ 필수값이 아니고 해당사항이 없는경우는 미선언 하시면 됩니다.
요청 XML node 반복 field field 설명 Data type format max 필수 비고
request <reqst>
request <apiQnaReplyReg>
request login_id 로그인ID <![CDATA[]]> Y
request company_id 고객사 아이디 Y shopPling 발급
request api_auth_key API 인증코드 Y shopPling 발급
request <qnaReplyInfo> Y
request Y qna_key QnA 번호 key Y
request Y reply 답변내용 <![CDATA[]]> Y
request </qnaReplyInfo> Y
request </apiQnaReplyReg>
request </reqst>
응답 XML node 반복 field field 설명 Data type format max 필수 비고
response <rspns>
response <apiQnaReplyRegRst>
response login_id 로그인ID <![CDATA[]]> Y
response company_id 고객사 아이디 Y shopPling 발급
response <qnaReplyListRst> Y
response Y qna_key QnA 번호 key Y
response Y code 성공,실패 결과코드
response Y msg 성공,실패 결과메세지 <![CDATA[]]>
response </qnaReplyListRst> Y
response succs_cnt 성공건수
response fail_cnt 실패건수
response </apiQnaReplyRegRst>
response </rspns>

XML 요청 예시

<?xml version="1.0" encoding="UTF-8"?>
<reqst>
   <apiQnaReplyReg>
      <login_id><![CDATA[shoppling]]></login_id>
      <company_id>S001</company_id>
      <api_auth_key>fgsddfgfkdgdkfgk</api_auth_key>
      <qnaReplyInfo>
         <qna_key>105987</qna_key>
         <reply><![CDATA[해당 상품은 품절되었습니다.]]></reply>
      </qnaReplyInfo>
      <qnaReplyInfo>
         <qna_key>105986</qna_key>
         <reply><![CDATA[해당 상품은 3일 후에 배송예정입니다.]]></reply>
      </qnaReplyInfo>
   </apiQnaReplyReg>
</reqst>

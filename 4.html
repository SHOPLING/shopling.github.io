쇼핑몰별 상품수정 요청 (마지막 수정일시: 2024-07-04 15:26:29)

기능 쇼핑몰별 상품수정 요청 (아래 둘중 하나 골라서 사용)
URL https://api.shopling.co.kr/prod/prod_each_mall_modify_api.phtml?mode=1&xml_src_url=(아래 xml 데이타가 있는 url 기술) https://api.shopling.co.kr/prod/prod_each_mall_modify_api.phtml?mode=2 -> 소켓통신으로 바로 xml로 밀어넣을시
※ 필수값이 아니고 해당사항이 없는경우는 미선언 하시면 됩니다.
요청 XML node 반복 field field 설명 Data type format max 필수 비고
request <reqst>
request <apiProdEachMdy>
request login_id 로그인ID <![CDATA[]]> Y
request company_id 고객사 아이디 Y shopPling 발급
request api_auth_key API 인증코드 Y shopPling 발급
request partner_id 필요시 partner_id (pdion, cl … , onchannel
request <goodsInfo> Y 상품 정보 시작 태그(상품당1개)
request Y mall_key 쇼핑몰 코드 Y
request Y goods_key 샵플링 상품코드 Y 존재하면 먼저 검색함 우선순의 첫번째 (수정 key)
request Y ptn_goods_cd 고객사 상품코드 <![CDATA[]]> Y 삽플링 상품코드가 없을시 검색함 우선순의 두번째 (수정key)
request Y sale_price 판매가 변수 미선언시 수정되지않음
request Y org_price 공급가 변수 미선언시 수정되지않음
request Y list_price 소비자가 변수 미선언시 수정되지않음
request Y margin 마진율 변수 미선언시 수정되지않음
request Y dtl_desc 상세설명 변수 미선언시 수정되지않음
request Y prod_nm 상품명 변수 미선언시 수정되지않음
request Y attri_cd_1 상품정보고시분류코드 <![CDATA[]]> (상품정보고시 분류코드 속성 변수 참조) 변수 미선언시 수정되지않음
request </goodsInfo> Y
request </apiProdEachMdy>
request </reqst>
응답 XML node 반복 field field 설명 Data type format max 필수 비고
response <rspns>
response <apiProdEachMdyRst>
response login_id 로그인ID <![CDATA[]]> Y
response company_id 고객사 아이디 Y shopPling 발급
response <goodsRst> Y
response Y goods_key 샵플링 상품코드
response Y ptn_goods_cd 고객사 상품코드 <![CDATA[]]>
response Y code 성공,실패 결과코드 <![CDATA[]]>
response Y msg 성공,실패 결과메세지 <![CDATA[]]>
response </goodsRst>
response succs_cnt 성공건수 실패시 에러코드 반환
response fail_cnt 실패건수 실패시 에러메세지 반환
response </apiProdEachMdyRst>
response </rspns>

주문 클레임요청 처리 (마지막 수정일시: 2023-08-22 14:23:03)

기능 삽플링 주문 클레임 처리 요청 하기 (아래 둘중 하나 골라서 사용)
URL https://api.shopling.co.kr/order/order_claim_req_api.phtml?mode=1&xml_src_url=(아래 xml 데이타가 있는 url 기술) https://api.shopling.co.kr/order/order_claim_req_api.phtml?mode=2 -> 소켓통신으로 바로 xml로 밀어넣을시
※ 필수값이 아니고 해당사항이 없는경우는 미선언 하시면 됩니다.

request Y claim_status 변경할 클레임 주문상태 <![CDATA[]]> Y 클레임 처리 가능 상태 : 
  신규주문,발송준비,발송보류,발송대기 → 취소접수
  취소접수 → 취소완료
  발송완료,주문출고완료 → 반품접수, 교환접수
  반품접수 → 반품수거완료
  교환접수 → 교환수거완료
request Y exchange_new 교환접수시 신규주문생성여부 <![CDATA[]]> Y 시 교환주문으로 신규주문 생성함
request Y mall_claim_rsn2 클레임 사유 <![CDATA[]]> 클레임 정보(클레임 아닐시 기재하지 않습니다.)
교환사유 : 색상변경, 사이즈변경, 기타,배송오류, 상품파손, 옵션변경, 상품불량, 상품불만족
반품사유 : 상품품절, 배송오류, 배송지연,단순변심, 기타, 상품불만족, 상품불량, 상품파손
취소사유 : 상품품절, 기타, 단순변심,배송지연, 상품변경
request Y memo 클레임내용 <![CDATA[]]> 클레임 정보(클레임 아닐시 기재하지 않습니다.)
request Y prod_use_status 상품사용상태 <![CDATA[]]> 클레임 정보(클레임 아닐시 기재하지 않습니다.)
알수없음, 미개봉,  개봉후미사용,   개봉후사용
request Y pickup_tp 반품,교환수거방법 <![CDATA[]]> 클레임 정보(클레임 아닐시 기재하지 않습니다.)
교환방법: 맞교환지시, 고객발송, 수거완료확인, 수거지시, 누락재출고, 전체재출고
반품방법: 수거지시, 고객발송, 수거완료확인, 쇼핑몰수거

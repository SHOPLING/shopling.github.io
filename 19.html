쇼핑몰 상품코드 가져오기 (마지막 수정일시: 2023-08-22 14:23:03)

기능 샵플링 상품코드에 해당하는 쇼핑몰 상품코드 가져오기 (아래 둘중 하나 골라서 사용)
URL https://api.shopling.co.kr/prod/get_mall_goods_cd.phtml?mode=1&xml_src_url=(아래 xml 데이타가 있는 url 기술) https://api.shopling.co.kr/prod/get_mall_goods_cd.phtml?mode=2 -> 소켓통신으로 바로 xml로 밀어넣을시
※ 필수값이 아니고 해당사항이 없는경우는 미선언 하시면 됩니다.
요청 XML node 반복 field field 설명 Data type format max 필수 비고
request <reqst>
request <apiGetMallGoodsCd>
request login_id 로그인ID <![CDATA[]]> Y
request company_id 고객사 아이디 Y shopPling 발급
request api_auth_key API 인증코드 Y shopPling 발급
request partner_id 파트너 코드 필요시 partner_id (habibi)
request <goodsInfo> Y
request Y mall_key 쇼핑몰 코드 <![CDATA[]]> Y 쇼핑몰 코드 필수
request Y mall_login_id 쇼핑몰 아이디 <![CDATA[]]> 쇼핑몰 로그인 아이디 선택
request Y goods_key 샵플링 상품코드 <![CDATA[]]> Y 샵플링 상품코드 필수
request </goodsInfo> Y
request </apiGetMallGoodsCd>
request </reqst>

response <rspns>
response <apiGetMallGoodsCdRst>
response login_id 로그인ID <![CDATA[]]> Y
response company_id 고객사 아이디 Y
response <goodsListRst> Y
response Y mall_key 쇼핑몰 코드
response Y mall_login_id 쇼핑몰 아이디 <![CDATA[]]>
response Y goods_key 샵플링 상품코드 <![CDATA[]]>
response Y mall_goods_cd 쇼핑몰 상품코드 <![CDATA[]]>
response Y link_seq 샵플링 연동번호 <![CDATA[]]>
response Y code 성공,실패 결과코드 <![CDATA[]]>
response Y msg 성공,실패 결과메세지 <![CDATA[]]>
response </goodsListRst> Y
response succs_cnt 성공건수
response fail_cnt 실패건수
response </apiGetMallGoodsCdRst>
response </rspns>

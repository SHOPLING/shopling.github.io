송장등록 (마지막 수정일시: 2023-08-22 14:23:03)

기능 삽플링에 송장등록 하기 (아래 둘중 하나 골라서 사용)
URL https://api.shopling.co.kr/invoice/invoice_registration_api.phtml?mode=1&xml_src_url=(아래 xml 데이타가 있는 url 기술) https://api.shopling.co.kr/invoice/invoice_registration_api.phtml?mode=2 -> 소켓통신으로 바로 xml로 밀어넣을시
※ 필수값이 아니고 해당사항이 없는경우는 미선언 하시면 됩니다.
요청 XML node 반복 field field 설명 Data type format max 필수 비고
request <reqst>
request <apiDlvyReg>
request login_id 로그인ID <![CDATA[]]> Y
request company_id 고객사 아이디 Y shopPling 발급
request api_auth_key API 인증코드 Y shopPling 발급
request <dlvyInfo> Y
request Y ord_no 샤플링 주문번호 Y
request Y dlvery_id 택배사코드 <![CDATA[]]> Y 택배사 코드 참조
request Y invoice_no 운송장번호 <![CDATA[]]> Y
request Y expect_dt 배송예정일 yyyymmdd
request </dlvyInfo> Y
request </apiDlvyReg>
request </reqst>
응답 XML node 반복 field field 설명 Data type format max 필수 비고
response <rspns>
response <apiDlvyRegRst>
response login_id 로그인ID <![CDATA[]]> Y
response company_id 고객사 아이디 Y shopPling 발급
response <dlvyListRst> Y
response Y ord_no 샤플링 주문번호 Y
response Y code 성공,실패 결과코드 <![CDATA[]]>
response Y msg 성공,실패 결과메세지 <![CDATA[]]>
response </dlvyListRst> Y
response succs_cnt 성공건수
response fail_cnt 실패건수
response </apiDlvyRegRst>
response </rspns>
